# SNAP CLASS – AI Attendance System

## Overview

SNAP CLASS is an AI-powered attendance management system that automates classroom attendance using facial recognition and voice recognition. It provides a fast, secure, and user-friendly alternative to traditional attendance methods, reducing manual effort while improving accuracy.

## Features

* Face Recognition–based student authentication
* Voice Recognition for attendance verification
* Teacher and Student dashboards
* Subject creation and enrollment using unique join codes
* Automatic attendance recording
* Attendance history and reports
* Secure login and registration
* Modern, responsive Streamlit interface
* Powered by Supabase cloud database

## Technologies Used

* Python
* Streamlit
* OpenCV
* NumPy
* Pandas
* Scikit-learn
* Resemblyzer
* Supabase
* Pillow

## Project Workflow

1. Teachers create subjects and share a unique join code.
2. Students register using facial and optional voice enrollment.
3. Students enroll in subjects using the join code.
4. Teachers capture classroom images or use voice attendance.
5. AI identifies students and records attendance automatically.
6. Attendance records are securely stored and displayed through dashboards.

## Benefits

* Eliminates manual attendance
* Saves classroom time
* Improves attendance accuracy
* Prevents proxy attendance
* Easy-to-use interface for teachers and students
* Secure cloud-based data storage
